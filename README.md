# Money-Coach
Mobile Development CSCI 3660 Exam 3 Assignment. Financial coach app for teens with LLM integration.

# Features
- LLM integration to explain financial concepts in simple language.
- Supply transaction history, set goals and budgets, daily reminders, budget limit and overspending warnings.
- Track income and expenses for a live running budget.
- Notifications settings and currency setting.
- Provide financial summaries in simple language with engaging visuals.
- Track multiple savings goals with custom names, icons, color, and progress indicators.


# Technology
React, Expo, Groq API
